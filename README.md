## 配置环境
1. "serve": "vue-cli-service serve --open", 自动打开页面
2. 给src配置别名
     "paths": {
      "@/*": [
        "src/*"
      ]
    },
3. 关闭语法检查
  lintOnSave:false
## 安装依赖
1. npm i element-ui -S   npm install babel-plugin-component -D
2. npm i vue-router@3
3. npm i vuex@3
4. npm i axios -S
5. npm i querystring -S
5. npm i normalize.css -S
6. npm i echarts -S
